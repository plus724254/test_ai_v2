## 完整遊戲流程

1. [ ] 開始遊戲 → 創建角色
2. [ ] 進入村莊 → 升級裝備
3. [ ] 進入森林 → 戰鬥敵人
4. [ ] 升級 → 進入黑暗森林
5. [ ] 打敗哥布林國王 → 進入更高級區域
6. [ ] 升級到足夠等級 → 進入魔王城
7. [ ] 與邪惡之神決戰 → 勝利

## 已知限制

- CSS最小化，主要使用內聯樣式
- 沒有音效或背景音樂
- 敵人AI簡單（固定攻擊模式）
- 沒有多人遊戲功能
- 存檔只在本地瀏覽器

## 測試環境

- 瀏覽器：Chrome/Firefox/Edge 最新版本
- 操作系統：Windows/Mac/Linux
- 分辨率：800x600 及以上
